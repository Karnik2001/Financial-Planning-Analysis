# Financial-Planning-Analysis

This project focuses on analyzing personal financial behavior, assessing financial risk, and predicting investment recommendation scores using data analytics and machine learning techniques.

## Key Features

Creation of personal finance KPIs:
- Savings Efficiency, Discretionary Income, Investment Capacity Index, Financial Stress Index

- Descriptive statistics and correlation analysis of key financial variables

- Risk classification using Debt-to-Income Ratio and Credit Score

Machine learning models for predicting Investment Recommendation Score:

- Linear Regression

- Polynomial Regression

- Decision Tree

- Random Forest

- Gradient Boosting

- AdaBoost

- ExtraTrees Regressor

## Tools & Technologies

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
